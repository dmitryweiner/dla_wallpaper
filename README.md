# DLA wallpaper
Drawing DLA tree on a Android live wallpaper

## Installation
Settings -> Display -> Wallpaper -> Live Wallpapers -> select "DLA tree live wallpaper"

[APK](https://github.com/dmitryweiner/dla_wallpaper/raw/master/app-debug.apk)
